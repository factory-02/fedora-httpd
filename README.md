# Информация

SPEC-файл для создания RPM-пакета **httpd**.

## Установка

1. Подключить репозиторий **METASTORE**: `dnf copr enable metastore/fedora-httpd`.
2. Установить пакет: `dnf install httpd`.

## Авторы

- [Kitsune Solar](https://kitsune.solar/) - мейнтейнер.
